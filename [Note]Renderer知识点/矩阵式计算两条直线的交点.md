

计算两条直线的交点的方法：    
一般式/点斜式    
矩阵式  

## 矩阵式

	a1*x + b1*y = c1
	a2*x + b2*y = c2

得到矩阵：  
[a1	b1]  *	[x] =	[c1]	
[a2 b2]		[y]		[c2]

**A** =	[a1	b1]				
1	--	[a2 b2]	

**X** =	[X]				
1	--	[Y]

**B** =	[c1]				
1	--	[c2]			

**A** * **X** = **B**  
**A^(-1)** * **A** * **X** = **A^(-1)** * **B**  
**A^(-1)** * **A** = 1  
**X** = **A^(-1)** * **B**  

所以只要会求 **A** 的逆，就能求出 **X**

请看 **《二阶矩阵求逆矩阵.md》**


也可以这样结合的求：

设A1 =   
[c1  b1]  
[c2  b2]  

设A2 =   
[a1  c1]  
[a2  c2]  

x = Det(A1)/Det(A)
y = Det(A2)/Det(A)

Det(A) 是 矩阵 **A** 的行列式。  
Det(A) = a1 * b2 - a2 * b1
